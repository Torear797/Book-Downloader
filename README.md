# Book Downloader

Swift-based инструмент для загрузки книг из Яндекс.Книги в формате EPUB

![Swift](https://img.shields.io/badge/Swift-5.5+-orange.svg)

## Особенности

- 📚 Загрузка книг из Яндекс.Книги

- 🔒 Обработка шифрованной метадаты с использованием AES-CBC

- 📖 Сохранение в EPUB формате

- 🎯 Command-line интерфейс

## Установка

Просто скачайте файл из каталога Artefacts

## Запуск

Запустите скачанный файл в своем терминале и передайте аргументы --bookid и --outdir.

--bookid: id нужной вам книги в сервисе Яндекс.Книги.
Пример: https://books.yandex.ru/books/CMsymD4Y
В данном случае id - CMsymD4Y

--outdir: каталог куда необходимо выгрузить скачанный файл. Необязательный аргумент.
По умолчанию книги создаются в загрузках в каталоге `books`

Пример команды:
``` bash
./Book_Downloader --bookid CMsymD4Y
